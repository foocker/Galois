# Statement

Conjecture A Fermat prime is a Fermat number \[ F_n = 2^{2^n } + 1 \] that is prime. The only known Fermat primes are F_0 =3,F_1=5,F_2=17,F_3 =257 ,F_4=65537 It is unknown if other fermat primes exist.
